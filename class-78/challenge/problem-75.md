# Problem: Temperature Converter (Celsius to Fahrenheit)

## Objective
Create a function that converts a temperature from Celsius to Fahrenheit.

## Scenario
You are developing a small feature for a weather application that needs to display 
temperatures in different units depending on the user’s preference.

The formula to convert Celsius to Fahrenheit is:

    F = (C * 9/5) + 32

## Requirements
1. Create a function named `convertToFahrenheit`.
2. The function must receive:
   - `celsius` (number)

3. The function should:
   - Apply the conversion formula
   - Return the temperature in Fahrenheit

## Example

### Input:
celsius = 0  

### Output:
fahrenheit = 32
